Integration Guide 

// Initialize Liquid Ether
const liquidEther = new LiquidEther({
    container: 'body',
    colorTheme: 'gold',
    intensity: 'medium',
    interactivity: true,
    particleCount: 60
});

// Change themes dynamically
liquidEther.setTheme('cyan');
liquidEther.setIntensity('high');

Simple HTML data attributes for auto-initialization: 

<body data-liquid-ether 
      data-liquid-ether-theme="gold" 
      data-liquid-ether-intensity="medium">
